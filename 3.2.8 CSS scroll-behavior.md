# CSS scroll-behavior

>兼容性还有些问题的

scroll-behavior:smooth 写在滚动容器元素上，可以让容器（非鼠标手势触发）的滚动变得平滑

凡是需要滚动的地方都加一句`scroll-behavior:smooth `就好了
